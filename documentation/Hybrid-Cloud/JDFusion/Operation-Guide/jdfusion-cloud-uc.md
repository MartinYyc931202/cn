
# 登录JDFusion控制台
## 1、准备好京东云账户
## 2、准备AK/SK
## 3、获取JDFuison产品邀请码
  获取邀请码参加【如何获取邀请码】说明
## 4、登录京东云控制台
### 访问[京东云官网](https://jdcloud.com)，【产品】-【混合云】-【JDFusion】，进入JDFusion产品页面
### 点击JDFuison产品页按钮[JDFusion控制台](uc.jdfusion.jdcloud.com)
### 输入京东云账户名和密码，页面跳转到JDFusion控制台页面邀请码页面
### 输入邀请码，点击确定
